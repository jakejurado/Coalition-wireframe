How to run this app:
1. download to your local environment.
2. with your terminal, go to the top folder of the app and run 'npm install'
3. once installed, run 'npm run dev'
4. open your webpage to localhost:3000.
5. compare what you see in the web page to the link below for Adobe wire frame
